# Hello World - LangChain Course

A simple introduction to LangChain demonstrating basic prompt templates and LLM integration.

## Overview

This project demonstrates how to:

- Create prompt templates with LangChain
- Connect to LLMs (OpenAI or Ollama)
- Chain prompts with language models
- Generate summaries and extract information

## AI Observability with LangSmith

- Monitor and trace LLM calls
- Debug prompt performance
- Track token usage and costs
- Analyze model responses

## Requirements

- Python 3.11+
- OpenAI API key (for ChatOpenAI) or Ollama (for ChatOllama)
- Environment variables configured via `.env` file
- LangSmith API key for observability

## Usage

Run the main script:

```bash
python main.py
```
